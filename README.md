**Поисковая система вакансий**

Учебный курсовой проект за 3 курс.
Веб-приложение для поиска вакансий и отклика соискателей.

---

*Функционал:*
- Регистрация и авторизация пользователей
- Создание вакансий
- Модерация вакансий
- Просмотр вакансий
- Создание резюме
- Отклик на вакансию, с прикреплением резюме и без него
- Просмотр откликов
- Одобрение или отклонение отклика

---

*Стек технологий*

Frontend:
- React
- JavaScript
- SCSS
- Axios
- MobX

Backend:
- Node.js
- Express
- Sequelize

DB:
- PostgreSQL

---

*Запуск приложения:*

Backend:
- cd server
- npm i
- npm run dev

Frontend:
- cd client
- npm i
- npm run dev
