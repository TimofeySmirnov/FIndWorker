# Поисковая система вакансий

Курсовой проект за 3 курс.
Веб-приложение для поиска вакансий и отклика соискателей.

---

## Функционал:

### Пользователи и доступ
- Ролевая модель: соискатель, работодатель, модератор
- Регистрация и авторизация пользователей
- Аутентификация на основе JWT

### Вакансии
- Создание вакансий работодателями
- Модерация вакансий
- Просмотр и фильтрация списка вакансий

### Резюме и отклики
- Создание резюме соискателями
- Отклик на вакансию с прикреплением резюме или без него
- Просмотр откликов на вакансии
- Одобрение или отклонение откликов работодателем
